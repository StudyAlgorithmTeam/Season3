EASY

-   _스터디 강제 하차_

HARD

-   2진아웃
-   3지각 = 1결석
-   불참 여부와 상관없이 추천문제와 코드 제출은 필수.
-   공결 사유로 인한 불참은 출석 인정 (필요에 따라 인민재판 있을 수 있음.)

-   풀어본 문제만 추천하기
-   제출 기한은 스터디 당일 00시 정각까지로 앞당김
